GraphLab PageRank

Installation Instructions

Compilation instructions via Graphlab.

Dataset

Nodes: 3997962

Edges: 34681189

Compressed size: 119M

Example single-node runtime: 15s

Link: SNAP (com-lj.ungraph.txt.gz)

Running

From the directory containing the gunzip'd dataset, " --graph=com-lj.ungraph.txt --format=snap --saveprefix=output" (e.g. "./graphlab/release/toolkits/graph_analytics/pagerank --graph=com-lj.ungraph.txt --format=snap --saveprefix=output") will run Graphlab's PageRank tool on the LiveJournal dataset, outputting each vertex's PageRank into a series of files into a series of files named 'output'.
