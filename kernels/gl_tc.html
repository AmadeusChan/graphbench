GraphLab Undirected Triangle Counting

Installation Instructions

Compilation instructions via Graphlab.

Dataset

Nodes: 3997962

Edges: 34681189

Compressed size: 119M

Example single-node runtime: 7.95s

Link: SNAP (com-lj.ungraph.txt.gz)

Running

From the directory containing the gunzip'd dataset, " --graph=com-lj.ungraph.txt --format=snap" will run Graphlab's undirected triangle count tool.
