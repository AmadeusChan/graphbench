Installation Instructions

Download and compilation instructions for Spark 0.9.0 via Spark.

Dataset

Nodes: 81306

Edges: 1768149

Compressed size: 10.1M

Example single-node runtime: 51s

Link: SNAP (twitter_combined.txt.gz)

Running

From Spark's base directory, "./bin/run-example org.apache.spark.examples.JavaPageRank local " (e.g. "./bin/run-example org.apache.spark.examples.JavaPageRank local ~/twitter_combined.txt 10") will run spark's example page rank code locally on the provided dataset for the specified number of iterations, outputting the results to stdout.
