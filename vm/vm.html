Graphbench Virtual Machine

Graphlab provides a VM image which comes pre-packedaged with reference kernel implementations.

In addition to the regular sequential runtimes, the VM also provides:

Spark:

  • PageRank

Graphlab:

  • PageRank
  • Triangle Counting

The VM image comes in .ova format compatible with the latest versions of VirtualBox and VMware Player. (Note: VMware Player may complain that the image "did not pass OVF specification performance" when you attempt to import. Simply hitting 'retry' will resolve the issue)
