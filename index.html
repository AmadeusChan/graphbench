Graphbench

A graph application benchmark suite

GraphBench is a suite of graph application benchmarks under development in the Sampa group at the University of Washington, Department of Computer Science and Engineering. Check back soon for updates!

Initial benchmarks include straw man implementations of the following graph kernels:

  • Breadth-First Search

    Starting from a specified node, create a tree, of minimum height, stretching from that node to all other reachable nodes in the graph. A simple benchmark to characterize the ability of the system to traverse the edges of a graph.

  • Single-Source Shortest Path

    Starting from a specified node in a graph with non-negative edge weights, for each other node in the graph, calculate minimum cost path from the origin to that node. For each node, it is sufficient the specify the 'previous' node on the least-cost path. Relevant for transportation network analysis.

  • Triangle Counting

    Count the triangles present in the graph. Relevant to applications which require the characterization of sub-graphs, such as social network and protein interaction analysis.
