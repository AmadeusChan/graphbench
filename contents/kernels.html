Initial benchmarks include single-threaded, straw man implementations of the following graph kernels:

  • Breadth-First Search

    Starting from a specified node, create a tree, of minimum height, stretching from that node to all other reachable nodes in the graph. A simple benchmark to characterize the ability of the application to traverse the edges of a graph.

  • Single-Source Shortest Path

    Starting from a specified node in a graph with non-negative edge weights, for each other node in the graph, calculate minimum cost path from the origin to that node. For each node, it is sufficient the specify the 'previous' node on the least-cost path. Relevant for transportation network analysis.

  • Triangle Counting

    Find the unique triangles in the graph. Relevant to applications which require the characterization of sub-graphs, such as social network and protein interaction analysis.
