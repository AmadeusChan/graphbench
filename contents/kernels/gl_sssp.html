GraphLab Single-Source Shortest Path

Installation Instructions

Compilation and use instructions on Graphlab's github.

Dataset

Nodes: 3997962

Edges: 34681189

Compressed size: 119M

Example single-node runtime: 7.95s

Link: SNAP (com-lj.ungraph.txt.gz)

Running

From the directory containing the gunzip'd dataset, " --graph=com-lj.ungraph.txt --format=snap --saveprefix=output --source=0" will run Graphlab's single-source shortest path tool on the LiveJournal dataset, using vertex 0 as the source, and outputting the resulting path costs to each other node in one or more files beginning with the prefix 'output'.
